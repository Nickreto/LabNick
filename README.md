# Задание 1 tree map

## Задача 1

### Текст задачи

Дерево содержит строки. Заменить в каждой первый символ на заданный.

### Алгоритм решения задачи

Ввод количества элементов
Проверка ввода
Создание списка из случайных строк
Вывод списка
Создание дерева из списка
Ввод символа для замены
Проверка символа
Замена на всех узлах
Вывод изменённого дерева

# Задание 2 tree fold

## Задача 1

### Текст задачи

Сформировать список из положительных листьев дерева (узел является
листом, если у него нет ни левого, ни правого поддерева).

### Алгоритм решения задачи

Ввод количества элементов
Проверка ввода
Создание списка из случайных строк
Вывод списка
Создание дерева из списка
Вывод дерева
Поиск листьев дерева
Вывод листьев списком
