# Липин Никита БАС1 Лабораторная No1

# Задание 1 

## Задача 1 

### Текст задачи

Меню в кафе
1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно). 2 запроса
должны возвращать перечень, 2 запроса одно значение.

### Алгоритм решения задачи

Класс Dish
 - конструкторы для пользователя и для приватного ввода внутренними функциями
 - Геттеры и сеттеры для всего кроме айди.
 - Айди только геттер
Класс BinarWorker
 - Метод добавления блюда
 - Метод удаления блюда
 - Методы LINQ:
    - веганские блюда
    - блюда в ценовом диапазоне
    - блюда по убыванию рейтинга
    - блюда по возрастанию цены
 - Метод сохранения меню в файл
 - Метод загрузки меню из файла
Класс Checker
 - Метод ввода int
 - Метод ввода double
 - Метод ввода char
 - Метод ввода string
